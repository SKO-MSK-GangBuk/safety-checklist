📋 현장 위험성 평가표 모바일 작성 도구 사용 가이드

본 도구는 매번 종이 서류를 들고 다니거나 수기로 작성하는 번거로움을 줄이기 위해, 모바일에서 항목을 입력하고 즉시 이미지로 저장할 수 있도록 제작되었습니다.

1. 접속 및 기본 정보 입력
• 접속: 제공된 링크로 접속합니다.
• 기본 정보: 협력사명, 도급사명, 일시, 국소명 등을 입력합니다.
• Tip: 날짜는 접속한 당일 날짜로 자동 세팅됩니다.

2. 점검 항목 체크 (중요)
• 기상 상태 / 장비 / 현장 특성 등: 해당하는 항목을 체크박스로 선택합니다.
• 기타(Etc) 입력:
• 리스트에 없는 항목이 있다면 '기타' 체크박스를 누르세요.
• 체크 시에만 별도의 텍스트 입력창이 나타납니다. 여기에 구체적인 내용을 적으면 양식의 '기타 ( )' 칸에 자동으로 입력됩니다.
• 실시간 미리보기: 화면을 아래로 내리면 내가 입력한 내용이 양식 위에 어떻게 찍히는지 실시간으로 확인할 수 있습니다.

3. 결과 이미지 저장 방법 (기기별)
📱 아이폰 (iOS) 사용자
1. 입력이 모두 완료되었는지 하단 미리보기 이미지를 확인합니다.
2. 하단의 생성된 이미지(미리보기)를 1~2초간 꾹 누릅니다.
3. 나타나는 메뉴에서 [사진 앱에 저장] 또는 **[이미지 저장]**을 선택합니다.
4. 사진첩(갤러리)에서 저장된 파일을 확인합니다.
🤖 안드로이드 및 PC 사용자
1. 입력 완료 후 하단의 [이미지 다운로드] 버튼을 클릭합니다.
2. 기기의 '다운로드' 폴더 또는 갤러리에 자동으로 저장됩니다.

⚠️ 주의사항 및 팁
• 오타 확인: 이미지가 생성된 후에는 수정이 불가능하므로 저장 전 미리보기에서 오타를 꼭 확인하세요.
• 새로고침: 페이지를 새로고침하면 입력했던 모든 데이터가 초기화됩니다. (임시저장 기능 없음)
• 이미지 해상도: 저장된 이미지는 실제 보고용으로 사용할 수 있는 고해상도 PNG 파일로 생성됩니다.
